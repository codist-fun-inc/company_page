# ステータス確認コマンド

プロジェクトの現在の状態を確認します。

## 実行手順

1. `git status` で Git の状態を確認
2. 最新のコミット情報を表示 (`git log -1 --oneline`)
3. Netlify デプロイ状況のリンクを表示

## 確認リンク

- 本番サイト: https://codist-company.netlify.app/
- デプロイ状況: https://app.netlify.com/sites/codist-company/deploys
- GitHub リポジトリ: https://github.com/codist-fun-inc/company_page
